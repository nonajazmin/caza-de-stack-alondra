# Vercel

## IDE

**Editor:** Visual Studio Code (VS Code)

Justificacion: Vercel y la mayoría de sus desarrolladores utilizan Visual Studio Code (VS Code) porque es un entorno de desarrollo ligero, gratuito y multiplataforma. Permite una integración directa con herramientas esenciales como ESLint y Prettier, que aseguran un código limpio, legible y estandarizado. Además, su integración con Vercel para VS Code facilita desplegar proyectos sin salir del editor, optimizando el flujo de trabajo y reduciendo errores humanos.

**Extensiones:**  
- ESLint  
- Prettier  
- Tailwind  
- GitHub Copilot  
- ChatGPT  

## Ecosistema de navegación

**Navegadores Chromium:**  
- Chrome  
- Firefox  
- Safari

Justificacion: Vercel trabaja con tecnologías web modernas, por lo que las pruebas de rendimiento y compatibilidad deben realizarse en los navegadores más usados: Google Chrome, Firefox y Safari. Esto garantiza que las aplicaciones creadas con Next.js (su framework principal) se vean y funcionen correctamente en distintos motores de renderizado (Blink, Gecko y WebKit). Probar en varios navegadores ayuda a ofrecer una experiencia de usuario consistente y de alto rendimiento.

## Gestión de Versiones

**Git y plataformas de hosting:**  
- GitHub  
- GitLab  
- Bitbucket

Justificacion: El uso de Git y plataformas como GitHub, GitLab o Bitbucket permite llevar un control detallado de los cambios en el código, trabajar de forma colaborativa y mantener versiones estables del proyecto. Vercel se integra automáticamente con estos servicios, lo que facilita el despliegue continuo (CI/CD): cada vez que se actualiza el repositorio, se genera una versión previa (preview deploy) para revisión antes de publicarla oficialmente.
Esto mejora la calidad del software y acelera el proceso de desarrollo.

## Diseño UI/UX y Prototipado

**Figma v0 by Vercel**

Justificacion: Antes de escribir código, los equipos de Vercel y los diseñadores frontend usan herramientas como Figma o v0 by Vercel para crear prototipos visuales. Esto permite definir la estructura, colores, interactividad y experiencia del usuario antes del desarrollo real.

## Lenguajes y Herramientas Base

- HTML  
- CSS3  
- JavaScript  
- TypeScript

Justificacion: Vercel se basa en tecnologías modernas del desarrollo web: HTML5, CSS3, JavaScript y TypeScript. Estas garantizan compatibilidad, rendimiento y escalabilidad en cualquier navegador.
Asimismo, herramientas como React y Next.js (creado por Vercel) permiten construir aplicaciones rápidas y optimizadas para SEO.

## Herramientas Modernas

**React, Tailwind CSS, Turborepo v0 by Vercel Parallel**

**Tailwind:** facilita crear interfaces atractivas con un sistema de clases utilitarias, mientras que Turborepo mejora la gestión de proyectos grandes o con múltiples aplicaciones.
En conjunto, este stack tecnológico representa eficiencia, velocidad y mantenibilidad.
